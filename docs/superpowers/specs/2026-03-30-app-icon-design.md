# AI Chat Note - App Icon 设计规格说明书 (Design Spec)

## 1. 设计背景与核心理念

### 1.1 产品定位
AI Chat Note 是一款旨在解决“AI对话多模型散乱”与“知识流失”问题的 Web 应用。其核心价值在于：**“聊天即生产，对话即沉淀”**，强调将散乱的 AI 沟通记录转化为结构化的知识笔记。

### 1.2 图标设计隐喻：知识地层 (Knowledge Layers)
基于“知识沉淀与积累”的核心动作，图标放弃了抽象的几何体，选择了**“三层实体笔记本堆叠”**的具象化表达。

- **三层叠加**：象征时间的推移和知识的累积（过去、近期、现在）。
- **实体笔记结构**：采用清晰的笔记本封面、书脊与纸张侧边，直白地传达出“笔记库/档案库”的产品属性。
- **色彩倒置 (顶蓝底紫)**：完美契合“沉淀”的过程。顶部明亮活力的科技蓝代表正在发生的 AI 活跃对话；底层深邃静谧的暗紫代表已被归档、内化的知识库。由动入静，由浅入深。

---

## 2. 视觉结构与层级

图标采用等距视角 (Isometric-like) 的正面错位叠加设计，共分为三个主要层级（从下至上）：

### 2.1 底层：深层记忆 (Deep Archive)
- **含义**：代表历史久远、已完全沉淀的底层知识库。
- **形态**：最底层的笔记本。
- **配色**：
  - 封面主色：深紫色 `#581C87`，带有一定的暗色透明度 (`opacity: 0.8`)。
  - 书脊边缘：极暗的紫黑色 `#3B0764`。
  - 书签带：亮紫色 `#A855F7`（作为点缀）。

### 2.2 中层：近期沉淀 (Recent Accumulation)
- **含义**：代表近期整理归档的笔记过渡层。
- **形态**：中间层的笔记本，位于底层与顶层之间起承上启下的作用。
- **配色**：
  - 封面主色：靛蓝色 `#4338CA` (`opacity: 0.9`)，处于蓝紫过渡态。
  - 书脊边缘：深靛蓝色 `#312E81`。

### 2.3 顶层：活跃焦点 (Active Focus)
- **含义**：代表当前正在进行的 AI 交互、正在生成与提取的新知识。
- **形态**：最顶层的笔记本，是视觉的绝对中心。带有代表内容的 UI 刻线。
- **配色**：
  - 封面主色：明亮的科技蓝 `#3B82F6`（无透明度，最实最亮）。
  - 书脊边缘：深蓝色 `#1D4ED8`。
  - UI 刻线：纯白 `#FFFFFF` 和浅蓝 `#BFDBFE`，附带发光节点 `#93C5FD`。
  - 书签带：亮蓝色 `#60A5FA`。

## 3. 材质与细节处理

### 3.1 柔和透光的纸张侧边
为了平衡“实体书本”的写实感与“AI 科技工具”的高级感，侧面的纸张边缘（通常是白色的部分）进行了特殊处理：
- **不使用纯白**：纯白会显得过于生硬和传统。
- **微透叠底处理**：
  - **顶层纸边**：高亮浅蓝色半透明 `rgba(219, 234, 254, 0.9)` 和 `rgba(191, 219, 254, 0.8)`。
  - **中层纸边**：柔和的靛青色半透明 `rgba(199, 210, 254, 0.6)` 和 `rgba(165, 180, 252, 0.5)`。
  - **底层纸边**：幽暗的淡紫色半透明 `rgba(216, 180, 254, 0.4)` 和 `rgba(192, 132, 252, 0.3)`。
- **效果**：呈现出类似磨砂玻璃或透光树脂的质感，厚实但兼具轻盈的科技数字感。

### 3.2 顶层 UI 刻线 (知识的具象化)
顶层蓝色的封面上必须包含极简的 UI 线条（类似 Markdown 或代码块），用来直接暗示这是“文本内容”和“AI 生成的信息”。
- 包含长短不一的横线组合，代表段落。
- 包含圆点节点，代表知识的结构化/知识图谱节点。

### 3.3 背景建议
由于图标本身是具有一定透明度和明度渐变的，建议放置在**深色背景**（如 `#0F0F16` 或 `#111827`）上以最大化其发光与叠底效果。如果必须作为应用图标放置在浅色背景上，需要为其增加一个深色的外框底板（如带圆角的深紫色方形底板）。
